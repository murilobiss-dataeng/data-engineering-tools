# Nathan - Currículo Profissional

Tudo nesta pasta. Site data-driven a partir de `resume.json`.

## Estrutura

```
nathan/
├── index.html
├── styles.css
├── script.js
├── resume.json     # Dados do currículo
├── perfil.jpg      # Foto (opcional)
├── *.pdf           # PDF do currículo
├── package.json
├── vercel.json
└── README.md
```

## Uso

1. Edite `resume.json` com suas informações
2. Adicione `perfil.jpg` (opcional)
3. Local: `npm run dev` → http://localhost:3000
4. Deploy Vercel: Root Directory = `curriculum-vitae/nathan`
